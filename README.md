## 👌 Quick Start

### Prerequisites
* [Node.js](https://nodejs.org/)
* [Git](https://git-scm.com/)

### Clone and Run

```bash
git clone https://github.com/divyadoredla/divya-sportfolio.git
cd divya-sportfolio
npm install
npm run dev
```

Your app will be available at: [http://localhost:5173](http://localhost:5173)
